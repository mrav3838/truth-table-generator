# Truth Table Generator Web Application

## Overview
A web-based application that generates truth tables for complex boolean expressions.

## Features
- Accepts logical expressions
- Generates full truth tables
- Web-based interface

## Tech Stack
- Java (logic engine)
- HTML, CSS, JavaScript
